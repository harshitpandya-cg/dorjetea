🍵 Dorje Tea — Website Clone
A pixel-perfect static clone of the Dorje Tea website, built with pure HTML and CSS. This project was made for frontend practice, focusing on layout, typography, and responsive design.

📄 Pages & Sections

Hero / Landing — Full-width banner with headline, tagline, and call-to-action
Menu / Products — Tea product listings with names, descriptions, and imagery
About Us — Brand story and values section
Contact / Footer — Contact details, social links, and footer layout

🏗️ Built With

HTML5 — Semantic markup and page structure
CSS3 — Styling, layout (Flexbox / Grid), and animations

No frameworks. No libraries. Just vanilla HTML and CSS.
